如何使用

在 Fiber 中配置静态资源服务非常简单，只需要一行代码：

app.Static("/public", "./app/public")

这行代码会将 ./app/public 目录下的所有文件作为静态资源，通过 /public 路径提供访问。

静态资源目录结构

您可以在 app/public 目录下按照以下结构组织您的静态资源：

app/public/
├── css/         # 样式表文件
├── js/          # JavaScript 文件
├── images/      # 图片资源
├── fonts/       # 字体文件
└── ...          # 其他静态资源

访问方式

假设您有一个 CSS 文件位于 app/public/css/style.css，您可以通过以下 URL 访问它：

/public/css/style.css
